## Name
luaconf - a C library for parsing config file which is written in lua.

## Status
This module has not been released yet.

## Synopsis

## System Requirements
* linux/freebsd 32/64bits
* lua-5.1

## Build
**step1**: get luaconf

    git clone https://github.com/portl4t/luaconf.git
    cd luaconf/src

**step2**: build

    make
    make install
